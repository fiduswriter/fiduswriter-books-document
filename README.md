# @fiduswriter/books-document

This package contains the Fidus Writer book-level importers and exporters.

## Usage

```javascript
import {FidusBookReader} from "@fiduswriter/books-document/importer/native"
import {NativeBookExporter} from "@fiduswriter/books-document/exporter/native"
import {HTMLBookExporter} from "@fiduswriter/books-document/exporter/html"
```

## Development

```bash
npm install
npm run build      # compile TypeScript to dist/
npm run typecheck  # run tsc --noEmit
npm test           # run the Jest test suite
```
